# Personal Portfolio Website

A clean, simple portfolio website built with HTML and CSS only.

## Files Included

- `index.html` - Main HTML file
- `style.css` - Stylesheet

## How to Customize

1. Open `index.html` and replace:
   - "Your Name" with your actual name
   - "Web Developer" with your title
   - About section text with your bio
   - Skills with your actual skills
   - Project cards with your real projects
   - Contact links with your email, GitHub, and LinkedIn

2. Open `style.css` to change colors:
   - Primary color: `#2563eb` (blue)
   - Gradient: `#667eea` to `#764ba2` (purple gradient)

## GitHub Pages Setup

### Step 1: Create Repository
1. Go to GitHub.com and sign in
2. Click "+" → "New repository"
3. Name it `yourusername.github.io` (replace with your GitHub username)
4. Make it Public
5. Click "Create repository"

### Step 2: Upload Files
1. Click "uploading an existing file"
2. Drag and drop `index.html` and `style.css`
3. Click "Commit changes"

### Step 3: Enable GitHub Pages
1. Go to Settings → Pages
2. Under "Source", select "main" branch
3. Click "Save"

### Step 4: Visit Your Site
- Your site will be live at: `https://yourusername.github.io`
- Wait 2-5 minutes for deployment

## That's It!

Your portfolio is now live on the internet.
